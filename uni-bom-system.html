Tải ứng dụng UNI BOM System

Ứng dụng quản lý BOM, công tác phí, chấm công nội bộ — cài một lần, dùng trên máy tính công ty.

Tải về cho Windows
Phiên bản 1.0.0 • Dung lượng — • Cập nhật —

Hướng dẫn cài đặt

1
Tải file cài đặt Bấm nút "Tải về cho Windows" phía trên, chọn nơi lưu file .exe.
2
Chạy file vừa tải Nhấp đúp vào file cài đặt, làm theo hướng dẫn trên màn hình. Quá trình cài mất khoảng 1 phút.
3
Đăng nhập Mở ứng dụng từ Desktop, đăng nhập bằng tài khoản công ty được IT/Quản trị viên cấp.
Windows cảnh báo "Unknown publisher"? Đây là cảnh báo bình thường vì ứng dụng nội bộ chưa đăng ký chứng chỉ số. Bấm "More info" (Thông tin thêm) rồi chọn "Run anyway" (Vẫn chạy) để tiếp tục cài đặt.
